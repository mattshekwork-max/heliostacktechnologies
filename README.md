# HelioStack Technologies

Parent company website for HelioStack Technologies LLC.

Live site:

`https://heliostacktechnologies.com/`

## Current Contents

This repository currently contains a static snapshot of the public site:

- `index.html`
- `_next/static/` assets generated by Next.js/Vercel
- site icons and font assets

The public page was originally generated by `v0.app` and deployed on Vercel.

## Local Preview

From this folder:

```bash
python3 -m http.server 8080
```

Then open:

`http://localhost:8080`

## Editing Note

This is a static site snapshot, not the original editable v0/Next.js source project.
For clean future development, export or recover the original v0/Next.js source and replace this snapshot with the actual app source.
